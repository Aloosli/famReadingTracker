# Screenshots

Drop screenshots here to bring the main README to life. **Capture them on a device where
emoji render properly** (an iPad or a Mac) — avatars and achievement patches are emoji, so
a machine without an emoji font will show empty boxes.

Save each shot with the filename below, then uncomment the matching `![...]` line in the
root `README.md` (in the Screenshots section).

| Filename                | What to capture                                                            |
| ----------------------- | -------------------------------------------------------------------------- |
| `profile-select.png`    | The "Who's reading tonight?" profile picker with all readers.              |
| `home-shelf.png`        | A reader's home page — the energy/goal bar and a book with progress.       |
| `add-book.png`          | The Add-a-book screen mid-search (or the barcode scanner).                 |
| `yearbook.png`          | The reading yearbook (`/year`) — the wrapped-style recap.                  |

Tips:
- Use a consistent theme (all light or all dark) across shots.
- A little sample data (a couple of books with progress, maybe an earned patch) looks far
  more inviting than an empty shelf.
- PNG keeps text crisp; aim for a portrait/tablet aspect ratio to match how it's used.
